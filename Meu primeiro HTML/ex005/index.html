Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica)

Nesta frase temos um termo em destaque usando a tag <strong> (semântica)

Itálico

Nesta frase temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase temos um termo em ênfase usando a tag <EM> (semântica)

Texto marcado

Podemos cria um texto marcado usandoa tag <mark>

Texto grande e texto pequeno

Nesta frase vemos um texto grande e um texto pequeno, ultilizando as tags <big> e <small>

Texrto deletado

Podemos marcar um texto como excluido para mostrar que pode ser lido mais não considerado

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi escrito depois

Tambêm podemos sublinhar o texto dando a mesma ênfase usando a tag <u>

Texto sobescrito

ultizamos o texto sobescrito em momentos específicos tipo: X 20 = 144, usando a tag <sup>

Texto subescrito

ultilizamos o texto subescrito nos seguintes casos: H2O , usando a tag <sub>
